# assets/references

Photos **réelles fournies par les mariés**. À déposer ici :

- `townhall-still.*` — **photo fixe de l'Hôtel de Ville de Dunkerque** (plan d'ouverture, large, ≥ 2560px). **Obligatoire.**
- `townhall-end-frame.*` — image caméra centrée sur la mairie (peut être extraite de la vidéo).
- `couple-photo.*` — photo(s) du couple (optionnel).

⚠️ **Aucune image de la mairie ne doit être générée ou inventée.** Voir `../../copy/asset-plan.md`.
