# assets/decorative

Décor **générique d'automne** (sans lien avec la mairie réelle) : feuilles, textures papier, filets/ornements dorés, monogramme des mariés.

- `leaves-*.svg|png` — feuilles d'automne discrètes.
- `paper-grain.*` — texture de fond subtile.
- `gold-rule.*`, `frame.*` — filets et cadres dorés.
- `monogram.svg` — initiales des mariés (optionnel).

✅ Seul endroit où un éventuel usage Higgsfield (après accord explicite) serait permis — **décor générique uniquement, jamais la mairie**. Voir `../../copy/higgsfield-video-prompt.md`.
